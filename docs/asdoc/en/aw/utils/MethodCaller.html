Class MethodCallerMethods
 
Packageaw.utils
Classpublic class MethodCaller

It’s a class calling for methods and functions from references directly, without calling Function.call() or Function.apply(). It contains hidden methods for calling functions by the number of arguments (the maximum number of arguments is 15), each of the methods contains the following code for calling a function from reference 
  static private function callNMethod(func:Function, a:Array):*{
   return func(a[0], a[1], a[2], a[3], ...a[N]);
  }
for calling a function by name of a class member 
  static private function callNByName(obj:Object, name:*, a:Array):*{
   return obj[name](a[0], a[1], a[2], a[3], ...a[N]);
  }
You can get access to a specific method according to the number of arguments with methods MethodCaller.getCaller() and MethodCaller.getCallerByMethod().
